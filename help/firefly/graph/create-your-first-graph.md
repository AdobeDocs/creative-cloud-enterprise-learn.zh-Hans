# 创建您的第一个图表

一旦知道节点、连接和模板是什么，您就可以开始构建自己的第一个工作流程。

1. **打开空白画布或从模板开始。** 浏览节点库或模板库以查找接近所需内容的起点。
2. **连接节点以生成工作流。** 从一个节点的输出到下一个节点的输入，绘制连接以把步骤链接在一起 — 例如，输入图像流进一个样式节点，然后进入一个变化生成器，最后进入导出。
3. **为以后要更改的任何内容添加输入控件。** 将静态值转换为滑块、拾色器或文本字段，以便您（或团队成员）可以调整关键参数，而无需编辑基础图表。
4. **运行图表并预览每个步骤。** 在完整工作流完成运行之前，可在任何节点预览输出。
5. **更改一个输入并重新运行。** 交换参考图像、调整样式提示或微调变量 — 所有下游内容都会自动更新。 没有什么是手工重建的。

## 首先尝试的功能

如果不确定从何处开始，请在模板库中打开一个现成的模板而不是空白画布 — 这是从头构建一个完整的工作图表之前查看完整的工作图表的最快方式。
