# 开始使用Firefly图

如果您以前使用过生成式AI工具，则可能会习惯键入提示、获得一个结果，并在下次需要类似工具时从头开始。 Firefly图的工作方式不同。 您构建的是图形，而不是单个提示：一个可视化、逐步的工作流程，其中连接并保存每个输入、变换和输出。 此页面将指导您完成构建第一个图形所需的一切：核心概念、分步第一个工作流程以及可立即打开和改编的现成模板库。

>[!TIP]
>
>把它想成菜谱，而不是一顿饭。 构建图表后，您可以将其提交，在新的输入上重新运行它，或更改一个步骤并使下游的所有内容自动更新。
