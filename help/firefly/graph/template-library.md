# Firefly图形模板库

每个模板都是有人已经构建的完整图表。 打开一个文件查看其具体工作原理，然后使用您自己的输入内容进行交换，使其成为您自己的文件。

| 模板 | 功能 | 链接 |
|---|---|---|
| **Bullet-Time VFX生成器** | 生成项目符号时间样式VFX序列。 | 打开模板 |
| **设计系统生成器** | 构建可重复使用的设计系统资源。 | 打开模板 |
| **Hero&#39;s Journey Story Builder** | 为故事板或活动弧线构建叙述顺序和故事结构。 | 打开模板 |
| **自定义实时着色器和画笔描边** | 创建自定义的实时着色器和画笔描边效果。 | 打开模板 |
| **素描到3D转折** | 将草图转换为3D环绕视图。 | 打开模板 |
| **目标颜色校正** | 以非破坏性方式应用选择性局部颜色校正。 | 打开模板 |
| **一致的头像** | 在组内生成一致的头像集。 | 打开模板 |
| **编辑摄影+换衣服** | 在拍摄时换上衣服，制作可编辑的照片外观。 | 打开模板 |

## 下一步

不确定哪个模板适合您的项目？ 首先了解核心概念，然后返回打开最近的匹配项。
