# &#x200B;5. 模板库

Firefly图形模板的快速引用索引，按每个模板的生产或操作进行组织。 每个示例都是一个起点 — 在生产中使用模板之前，替换您自己的品牌、产品和提示。

## 图像生成和样式

* [**入门 — 生成映像**](/help/firefly/graph/templates/get-started-gen-image.md) — 最简单的图形：一个提示节点进入一个生成节点到一个输出中。 将其用作与任何全新的用户一起打开的第一个模板。
* [**一致的角色生成**](/help/firefly/graph/templates/character-gen.md) — 加载角色的一个参考图像，然后为每个新拍摄互换场景或姿势提示。 当周围的场景发生变化时，角色引用会保持锁定状态。
* [**样式提取**](/help/firefly/graph/templates/style-extraction.md) — 在一个批准的参考图像中馈送。 样式节点提取其颜色、亮度和纹理处理，然后将该处理应用于任何通过同一图形运行的新图像。
* [**日落氛围**](/help/firefly/graph/templates/sunset-vibes.md) — 了解如何创建从文本提示中使用单词“Vibe”的3D排版图像。 模板会自动处理放置和色彩平衡。
