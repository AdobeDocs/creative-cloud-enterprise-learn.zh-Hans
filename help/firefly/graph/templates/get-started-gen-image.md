# 快速入门 — 生成图像

使用最简单的图了解基本知识：一个提示节点到一个生成节点到一个输出中。 [打开快速入门 — 生成图像模板](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:f556b557-d31b-4659-b9d1-358ed567c61d)。

>[!TIP]
>
>**开始之前** — 为获得最佳效果，请根据您自己的品牌、产品和工作流程自定义此模板。 在使用任何输出之前，交换参考图像、提示和副本。

![生成图像](../../assets/get-started-gen-image.png){align="center"}

[!BADGE 用例]{type=Informative tooltip="使用案例"}

* **零售** — 从简报生成第一个产品英雄图像，以便在接触真正的营销活动资源之前了解基本节点流程。
* **运行状况** — 在缩放到完整内容日历之前，在占位符产品拍摄上测试最简单的图像生成流程。
* **教育** — 在分配实际项目工作之前，生成第一个示例图像以向新的团队成员演示图形。
