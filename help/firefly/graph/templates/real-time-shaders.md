# 实时着色器

了解如何从图像开始应用三种不同的自定义着色器并实时预览结果。 直接在节点上调整着色器参数，而不是从头开始重新渲染。 [打开实时着色器模板](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:US:51848c67-e360-5c40-8eff-8517b3395d54)。

>[!TIP]
>
>**开始之前** — 为获得最佳效果，请根据您自己的品牌、产品和工作流程自定义此模板。 在使用任何输出之前，交换参考图像、提示和副本。

![实时着色器](../../assets/real-time-shaders.png){align="center"}

[!BADGE 用例]{type=Informative tooltip="使用案例"}

* **技术** — 为交互式展览演示中使用的3D产品配置器构建自定义风格化着色器。
* **汽车** — 在物理原型存在之前，在车辆模型上预览自定义绘画和材质着色器。
* **零售** — 在3D产品渲染上测试风格化素材外观，以便用于数字货架显示器。
