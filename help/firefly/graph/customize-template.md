# &#x200B;4. 自定义模板

每个模板都附带示例输入和通用用例。 仅当它反映您自己的摘要、资源和结果后，它才能成为适合&#x200B;*您*&#x200B;的工作流程。

1. **将示例输入替换为您自己的资源或简介。**
2. **重命名任何仍使用占位符文本的节点或标签。**
3. **在依赖结果之前，完全运行一次图形。**
4. **更改一个变量并重新运行它**，以便您可以确切地看到该步骤控制哪些内容。
5. **使用自定义版本自己的名称保存自定义版本。**
