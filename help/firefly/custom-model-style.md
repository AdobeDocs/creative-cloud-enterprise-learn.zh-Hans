# 使用样式创建自定义模型

了解如何在Firefly中构建自定义模型，以使用特定类型的样式（如背景、颜色和色相）为公司品牌创作全新的图像。 自定义模型允许您大规模创建相关的个性化内容。 尝试Adobe Firefly [此处](https://firefly.adobe.com/).
