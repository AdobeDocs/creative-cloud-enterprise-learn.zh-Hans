# Adobe [!DNL Firefly] 概述

![Firefly英雄图像](../assets/firefly.png)

萤火虫是新一代有创造力的AI模型，即将推出Adobe产品，最初侧重于图像和文本效果的生成。 Firefly提供了新方式来创意、创建和沟通，同时显著改善了创意工作流程。
