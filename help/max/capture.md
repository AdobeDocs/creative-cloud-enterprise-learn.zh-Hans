# [!DNL Capture]

Adobe[!DNL Capture]将手机和平板电脑变为设计收藏集工具。 用户可以创建许多不同类型的资源（图稿的单个数字元素）。   这些资源会自动与其他桌面和移动Adobe应用程序同步。 用户可以将其导入到其创意项目中，或轻松地与协作者共享。

**描述**
将移动设备上的图像和视频转换为适合您所有设计的创意构建块。

在本教程中，您将了解如何：
* 随时随地设计
* 通过CC Libraries与桌面应用程序集成
