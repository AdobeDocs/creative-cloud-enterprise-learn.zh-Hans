# [!DNL Capture]

![教程主图](../assets/Capture.jpg)

Adobe [!DNL Capture] 将手机和平板电脑转变为设计收藏工具。 用户可以创建许多不同类型的资源（图稿的单个数字元素）。   这些资源会自动与其他桌面和移动Adobe应用程序同步。 用户可以将其导入到其创意项目中，或轻松地与协作者共享。

在本教程中，您将了解如何：
