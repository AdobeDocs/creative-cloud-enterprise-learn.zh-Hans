# [!DNL Capture]

![教程英雄图像](../assets/Capture.jpg)

Adobe [!DNL Capture]将手机和平板电脑转变为设计收集工具。 用户可以创建多种不同类型的资源（图稿的单个数字元素）。   这些资源会自动与其他桌面和移动Adobe应用程序同步。 用户可以将它们导入其创意项目或轻松与协作者共享。
