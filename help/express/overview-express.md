# Adobe [!DNL Express] 概述

使用新的多功能Adobe Express应用程序设计作品集和TikTok视频、传单、简历、横幅、徽标等，以便快速轻松地创建内容。 利用由Adobe Firefly支持的生成式AI功能，创作引人注目的作品。
